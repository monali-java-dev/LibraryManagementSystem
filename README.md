# 📚 Library Management System (Java)

**Developed during:** Java Development Internship at [Codveda Technology](https://www.codveda.com)

---

## 💻 Project Overview
A Java-based desktop application to efficiently manage books, members, and library transactions. This system helps automate library operations and maintain records digitally.

---

## 🔧 Features
- ✅ Add, update, and delete books  
- ✅ Add and manage library members  
- ✅ Issue and return books with date tracking  
- ✅ Search books and members quickly  
- ✅ Generate reports for issued and overdue books  

---

## 🧠 Skills & Concepts Used
- Java programming  
- GUI development using **Java Swing**  
- Database management using **JDBC** (MySQL/PostgreSQL)  
- Exception Handling and Input Validation
